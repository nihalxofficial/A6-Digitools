💡 The Idea Behind DigiTools
DigiTools was born from one question — why does shopping for digital tools feel so clunky?
It's a subscription-based product platform built entirely with React.js, focused on making browsing, buying, and cart management genuinely enjoyable. Every layout, interaction, and detail was translated from polished Figma designs into a fast, responsive experience — no bloat, no confusion.
From a quick mobile browse to a full desktop deep-dive, DigiTools keeps up.

✨ Core Features
🗂️ Product Browsing & Cart Management

Browse products laid out in a structured 3-column grid
Add or remove any item from the cart with zero friction
Proceed to Checkout wipes the cart clean and resets the session

📌 Smart Navbar

Always shows the exact number of items currently in your cart
Cart count refreshes instantly with every add or remove action — no page reload needed

📐 Responsive by Default

Polished and functional on mobile, tablet, and desktop
UI built closely following Figma design specs for a consistent look and feel

🎯 Bonus Interactions

Tapping any product card bumps the cart counter in the Navbar right away
"Buy Now" fires a smooth toast alert powered by React-Toastify
Contextual notifications for every cart action — add, remove, and checkout


⚡ Tech Stack
Tool	What it does here
React.js	Drives the entire frontend UI
Tailwind CSS + DaisyUI	Keeps styling consistent and component-ready
JavaScript (ES6+)	Handles all logic and interactivity
React-Toastify	Delivers slick cart notifications
JSON	Manages and serves all product data
Features-- 1.Product Browsing & Cart Management.

2.Users can view products in a 3-column layout. Add or remove items from the cart. Proceed to checkout clears all items.

3.Dynamic Navbar--- Displays current cart item count. Cart icon updates dynamically when products are added or removed.

4.Responsive Design-- Fully responsive across mobile, tablet, and desktop. Clean, modern UI inspired by Figma designs.

Optional Features Implemented
